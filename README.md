# KySports Foundation Theme

A custom WordPress Block Theme built exclusively for the KySports Foundation.

## Mission

KySports empowers Kentucky's youth through sports media education, broadcasting, photography, videography, journalism, mentorship, and career development.

This theme is designed to provide a modern, fast, accessible, and scalable foundation for the organization's digital presence.

---

## Features

- Modern WordPress Block Theme
- Fully Responsive
- Video Homepage Hero
- Dynamic Homepage Sections
- Custom Post Types
- Student Stories
- Events
- Programs
- Partners
- Scholarship Support
- SEO Optimized
- Accessibility Focused
- Performance Optimized
